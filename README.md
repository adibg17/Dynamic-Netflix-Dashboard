# üé¨ Dynamic Netflix Dashboard (Tableau)

This project presents an interactive **Netflix content dashboard** built using **Tableau**. It provides insights into Netflix‚Äôs vast content library, exploring trends across countries, genres, ratings, and content types.

---

## üìÇ Project Overview

The dataset was sourced from a publicly available Netflix dataset and cleaned using **Excel**. Tableau was used to build a dynamic and visually appealing dashboard that allows users to explore patterns in Netflix‚Äôs global content distribution, genre trends, and maturity ratings.

---

## üìä Dashboard Features

### üîπ Interactive Title Search
- Filter any movie or TV show title.
- View detailed description, release year, duration, and genre instantly.

### üîπ Content Distribution by Country
- World map visualization showing the total number of titles available by country.
- Helps analyze Netflix‚Äôs regional content strategy.

### üîπ Maturity Ratings Breakdown
- Bar chart showing how content is distributed across different maturity levels.
- Useful to understand age-appropriateness of content.

### üîπ Movies vs TV Shows Split
- Bubble chart representing the ratio between movies and TV shows.
- Highlights that movies make up ~68% of Netflix content.

### üîπ Genre Popularity
- Horizontal bar chart showing the **Top 10 most common genres**.
- Combines multiple genre tags for richer insights.

### üîπ Time-Based Trends
- Area chart visualizing the total content added year-over-year.
- Differentiates between movies and TV shows to spot content growth trends.
